> # E-Commerce Web Application Using ASP.NET Core MVC
### A modern e-commerce web application built with ASP.NET Core 8.0 MVC, featuring product catalog browsing, shopping cart functionality, and secure Stripe payment integration. This frontend application provides an intuitive user interface for customers and administrators to interact with the e-commerce platform.

> # **ðŸ”¹ Key Features:**
### **1. User Authentication & Authorization**
- **Customer Registration & Login:** Secure authentication with session-based JWT token storage.
- **Role-Based Access Control:**
  - **Customer:** Browse products, manage cart, place orders, and view order history.
  - **Admin:** Full access to manage products, categories, and view all orders.
  - **Guest:** Browse products without authentication.
- **Profile Management:** Update personal details and track order status.

### **2. Product Catalog & Browsing**
- **Product Discovery:**
  - Browse products with advanced search, filtering, and sorting capabilities.
  - Category-based navigation (Electronics, Clothing, Books, Home & Garden, Sports).
  - Pagination for smooth browsing experience.
- **Product Details:**
  - Comprehensive product information including images, descriptions, pricing, and availability.
  - Category organization for easy navigation.

### **3. Shopping Cart System**
- **Cart Management:**
  - Add, update, and remove items from cart.
  - Real-time price calculations and subtotal updates.
  - Cart persistence across sessions.
- **Intuitive Interface:**
  - Easy quantity adjustments and item removal.
  - Clear display of cart totals and item details.

### **4. Secure Checkout & Payment Processing**
- **Stripe Integration:**
  - Secure payment processing through Stripe.
  - Support for credit/debit card payments.
  - Real-time payment validation and authorization.
- **Checkout Flow:**
  - Streamlined checkout process with order summary.
  - Automatic order confirmation upon successful payment.
  - Secure handling of payment information.

### **5. Order Management & Tracking**
- **Customer Order Tracking:**
  - View complete order history with detailed information.
  - Real-time order status updates (Processing, Shipped, Delivered).
  - Order details including items, quantities, and total amounts.
- **Admin Order Management:**
  - View and manage all customer orders.
  - Update order statuses and track fulfillment.

### **6. Admin Control Panel**
- **Product Management:** Full CRUD operations for products and categories.
- **Order Overview:** Monitor and manage customer orders.
- **Inventory Control:** Update product availability and stock information.

### **7. Responsive Design & User Experience**
- **Bootstrap 5 Integration:** Mobile-friendly, responsive design for all devices.
- **Modern UI/UX:** Clean, intuitive interface with smooth interactions.
- **Client-Side Interactivity:** jQuery-powered dynamic features for enhanced user experience.

### **8. Performance & Optimization**
- **API Integration:** Seamless communication with backend REST API services.
- **Structured Logging:** Serilog implementation for debugging and monitoring.
- **Clean Architecture:** Repository pattern and service layer for maintainability.
- **AutoMapper:** Efficient object-to-object mapping for ViewModels.

### **9. Security Features**
- **JWT Token Management:** Secure session-based authentication.
- **Input Validation:** FluentValidation for robust data validation.
- **Role-Based Authorization:** Protected routes and actions based on user roles.
- **Secure Payment Handling:** All payment data processed through Stripe's secure infrastructure.

> # **ðŸ”¹ Getting Started**

### **Prerequisites**
- .NET 8.0 SDK
- SQL Server (LocalDB or full instance)
- Stripe Account

### **Configuration**
1. Update connection string in `appsettings.json`
2. Configure Stripe API keys (Publishable Key, Secret Key, Webhook Secret)
3. Set application URL for redirects

### **Run the Application**
```bash
cd MVCApp
dotnet restore
dotnet run
```

Access the application at `https://localhost:7001`

### **Default Credentials**
- **Admin:** admin@ecommerce.com / Admin123!
